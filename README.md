# Supiki Mod

## プロジェクト概要
Minecraft Forge向けに、友好Mob「スピキ（Supiki）」を追加するためのModプロジェクトです。

## 開発環境
- Java 21
- Minecraft Forge 1.21.11（61.x）
- Gradle

## ビルド方法
```bash
./gradlew build
```
