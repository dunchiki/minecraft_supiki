# Supiki Mod

## プロジェクト概要
Minecraft Forge向けに、友好Mob「スピキ（Supiki）」を追加するためのModプロジェクトです。

## 開発環境
- Java 17
- Minecraft Forge 1.20.1（47.x）
- Gradle

## ビルド方法
```bash
./gradlew build
```
